== Description ==

Buddypress Follow Me is fork of BuddyPress Follow plugin (http://wordpress.org/extend/plugins/buddypress-followers/). buddypress-followers is now updated to the last version of Buddypress.

In order to keep the same community, i was decide to pull out Buddypress Follow Me and i invite all users of this plugin to install the main plugin BuddyPress Follow from http://wordpress.org/extend/plugins/buddypress-followers/.



Thank you for your understanding.
